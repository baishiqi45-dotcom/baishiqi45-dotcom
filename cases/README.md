final acceptance truth.

## Case Format

Each case should answer:

- original intent;
- where the project was losing control;
- control method used;
- what was handed off;
- how the artifact was reviewed;
- what became reusable;
- public boundary;
- next public artifact.

## Non-Claims

These case tracks do not publish private scripts, raw media, provider execution
state, account state, customer material, or final project acceptance.
