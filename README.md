# Hi, I build small repo-native tools for AI agent review.

I care about one boring but important thing: when an agent says "done", can another human or agent check what that confidence is based on?

Now shipping:

- [`epistemic-os`](https://github.com/baishiqi45-dotcom/epistemic-os): a claim-scope checker that stops weak evidence from becoming confident AI claims.

Planned kits:

- `freshness-patrol-kit`: official-first freshness checks for AI knowledge bases.
- `direction-card-proof-kit`: synthetic A/B proof scaffolds for human-reviewable creative AI outputs.
- `agent-clinic`: project handoff packets for repo-native AI agents.

## What I Care About

AI agents should not only answer faster. They should leave reviewable evidence:

- what changed;
- what was checked;
- what remains unproven;
- what needs human review;
- what should not be claimed yet.

## Current Line

I am building small open-source tools around verifiable AI workflows:

- claim scope;
- freshness checks;
- reviewable evidence;
- human-in-the-loop agent work.
