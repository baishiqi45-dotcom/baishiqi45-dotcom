# Video Admission Ladder

Use this when a video artifact exists but its review status is unclear.

## Artifact

```text
Artifact name:
Owner:
Source location:
Generated or captured:
Review span:
```

## Ladder

| Level | Evidence | Present? | Notes |
| --- | --- | --- | --- |
| Raw artifact | File, URL, export, manifest |  |  |
| Static evidence | frames, contact sheet, timestamps |  |  |
| Playback evidence | human watched relevant span |  |  |
| Relation evidence | before/after continuity checked |  |  |
| Owner seal | owner accepted scope |  |  |

## Decision

```text
Candidate only:
Admitted to edit review:
Admitted to story review:
Rejected:
Needs retake:
```

## Boundary

Static frames, model summaries, and sidecar comments are not final video
acceptance. Owner acceptance must stay explicit.
