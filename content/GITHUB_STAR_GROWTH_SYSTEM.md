# GitHub Star Growth System

This is the ethical growth system for the public method repository. The goal is
more qualified discovery, more useful bookmarks, and more serious conversations.
No bought stars, no fake accounts, no spam, no private-asset bait.

## Positioning Routes

### 1. AI Video Control Reference

Audience:

```text
AI video creators, short-drama teams, commercial content teams.
```

Star trigger:

```text
They find a continuity lock, retake rule, or admission gate they can copy today.
```

Proof needed:

- synthetic script-to-package case;
- continuity lock template;
- short teardown posts.

### 2. Non-Engineer Project Clinic

Audience:

```text
founders, operators, creators, PMs, solo builders.
```

Star trigger:

```text
They realize the startup packet can prevent a wrong first agent task.
```

Proof needed:

- Project Clinic startup packet;
- synthetic before/after example;
- issue template for public-safe clinic requests.

### 3. Agent Workflow Guardrail Library

Audience:

```text
people using Codex, Claude Code, Kimi Code, sidecars, or multi-agent workflows.
```

Star trigger:

```text
They need a vocabulary for evidence, handoff, review, and non-claims.
```

Proof needed:

- video admission ladder;
- outcome-delta writeback note;
- sidecar-as-advisor explanation.

## Weekly Operating Rhythm

Monday:

```text
ship one small method/template improvement
```

Wednesday:

```text
publish one short post linking to the relevant GitHub asset
```

Friday:

```text
open or close one public issue that shows active maintenance
```

Sunday:

```text
review stars, referrals, issues, and which asset caused attention
```

## Metrics

Track weekly:

- repository stars;
- profile repository views and clones if available;
- Pages visits if analytics are later added;
- issue opens;
- template clicks from posts;
- which content topic caused the best qualified conversation.

## CTA Rules

Good:

```text
If this template saved you a false start, star the repository so other operators
can find it.
```

Bad:

```text
Help me get stars.
```

Good:

```text
Use this as a bookmark for AI video control and Project Clinic templates.
```

Bad:

```text
Star because the project deserves attention.
```
