# Security Policy

This repository is a public method library. It does not operate production
runtime, provider accounts, billing, private datasets, or customer systems.

## Supported Scope

Report issues that affect this public repository, including:

- accidental exposure of private material in public files;
- broken public-safe boundary language;
- unsafe issue template wording;
- repository configuration that could expose secrets.

## Out Of Scope

- Provider execution, billing, credits, accounts, or job state.
- Private project workspaces and owner-controlled assets.
- Final artifact acceptance or commercial clearance.
- Requests to inspect private scripts, raw media, prompts, screenshots, or
  credentials in public issues.

## Reporting

Do not paste secrets or private material into a public issue. Open a minimal
public issue only if the problem can be described safely. Otherwise, keep the
material private with the project owner.
