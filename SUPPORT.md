# Support

Use GitHub issues only for public-safe method discussion.

## Good Public Issues

- A synthetic AI video continuity problem.
- A template request that can be discussed without private material.
- A Project Clinic routing question with public-safe context.
- A broken link, unclear method card, or missing example.

## Do Not Post Publicly

- Private scripts, raw media, prompts, screenshots, provider state, account
  material, customer information, local workspace paths, or final acceptance
  truth.
- Commercially sensitive project context.
- Security reports or credentials.

## Private Work

This public repository does not accept private project materials. Real project
assets, runtime implementation, provider operation, billing, credentials, and
final acceptance remain with the project owner.
